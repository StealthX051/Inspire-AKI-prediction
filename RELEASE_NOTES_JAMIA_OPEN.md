# JAMIA Open reproducibility release

This release archives the analysis code for the accepted JAMIA Open article:

**Integration of Intraoperative Data in Interpretable Machine Learning Models to Predict Postoperative AKI in Noncardiac Surgery Patients**

This release includes code, configuration files, tests, and environment files for reproducing the analyses after obtaining authorized access to INSPIRE version 1.3 from PhysioNet.

This release does not include INSPIRE source files, row-level derived datasets, train/test split files, patient- or operation-level predictions, patient-level SHAP values, generated reports, or trained model artifacts.

Source data:

- INSPIRE version 1.3, PhysioNet, doi:10.13026/46m4-f655

Suggested tag:

- `v1.0.0-jamia-open`
